Notes:
	The video spot tracker program requires DirectX version 9.0a or later(www.microsoft.com/directx).  It requires the camera drivers for any attached Roper or SPOT camera to be used.
	This program makes use of the ImageMagick libraries to open image files.
	This program makes use of the FFPMEG library to read video files.
